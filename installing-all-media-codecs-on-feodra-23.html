Installing all media codecs on Feodra 23
Sun 28 February 2016 linux fedora / command line / codec

The problem started as soon as I installed the Kazam. Kazam is a great desktop / mic recorder for linux systems but unfortunately it is not part the Fedoras repository. I obtained the rpm from the net and installed it but it lacked MP4 encoding.

The solution relies in installing more codec libraries on the system. At first you have to add the rpmfusion repositories using this line of code:

su -c 'dnf install http://download1.rpmfusion.org/free/fedora/rpmfusion-free-release-$(rpm -E %fedora).noarch.rpm http://download1.rpmfusion.org/nonfree/fedora/rpmfusion-nonfree-release-$(rpm -E %fedora).noarch.rpm'

And then install the full pack of vide codecs using this command:

dnf -y install gstreamer-plugins-bad gstreamer-plugins-bad-free-extras gstreamer-plugins-bad-nonfree gstreamer-plugins-ugly gstreamer-ffmpeg  gstreamer1-libav gstreamer1-plugins-bad-free-extras gstreamer1-plugins-bad-freeworld gstreamer1-plugins-base-tools gstreamer1-plugins-good-extras gstreamer1-plugins-ugly gstreamer1-plugins-bad-free gstreamer1-plugins-good gstreamer1-plugins-base gstreamer1

Annnd.... Done! Now Kazam can export files to mp4!

If you are just looking for a vide player, vlc is a great option. Just install it after the rpm fusion is enabled:

dnf -y install vlc
